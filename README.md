Toast-E Firmware
========== 

This repository comprises 2 main functions:
- Primary Logic Controller
- Python GATT (Bluetooth LE) server

Usage 
----- 
### Dependencies

Install Python requirements in primary or virtual environment:
```pip install -r requirements.txt```
